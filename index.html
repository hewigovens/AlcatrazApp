Introducing AlcatrazApp

Alcatraz is the Package manager for Xcode. Its self is originally designed as a Xcode plugin. But when Apple releases new version of Xcode which DVTPlugInCompatibilityUUIDs maybe also changed to some new value. It then causes all Xcode plugins - even Alcatraz - can not be loaded and this issue was been discussed long long ago.

AlcatrazApp is simply a standalone container app for Alcatraz. It loads Alcatraz just like Xcode without checking DVTPlugInCompatibilityUUIDs. So you can still manage your plugins without openning Xcode. sometimes It's really helpful.
